# Homebrew Tap for Rune

Homebrew formulae for [Rune](https://github.com/chasey-myagi/rune) — define functions, get APIs + workflows + distributed execution.

## Install

```bash
brew install chasey-myagi/tap/rune
```

Or tap first, then install:

```bash
brew tap chasey-myagi/tap
brew install rune
```

## Upgrade

```bash
brew upgrade rune
```

## What gets installed

- `rune` — CLI tool for managing Rune Runtime and calling Runes
- `rune-server` — Rune Runtime server (HTTP + gRPC)
